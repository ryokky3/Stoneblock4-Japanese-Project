# 概要

このプロジェクトは、Stoneblock4の敷居をより低くしようという試みによって開設されたプロジェクトです。  
今までの乱雑な翻訳とは違い、名詞(StoneBlock4など)を翻訳せず、日本語話者が自然に理解できるような文を目指しました。  
目指している翻訳は、下の通りです。  

・名詞は翻訳しない → クエストラインだけ翻訳することを前提とするため、アイテムの名詞などは極力翻訳しない。  
・装飾はいじらない → &rなどの装飾はいじらない。英語と日本語の差異は、文字だけにする。  
・スタイルを整える → 1を英数字にするか漢数字にするかを決めるなど。日本語に翻訳できない部分は、すこし変える。  

## 詳しい定義

・数字は漢数字ではなく英数字を使う。  
・アイテム名、ディメンション名、構造物名は何もいじらない。  
・基本的に敬語を使う。敬語が適さないと感じた場合、それは絶対条件ではない。  

## 導入について

この中に入っているftbquests\quests\lang\en_us.snbtをminecraft\config\ftbquests\quests\langに上書きで配置してください。

・個人で翻訳しているため、翻訳内容にミスやマニュアル通りに翻訳されていない可能性があります。もしミスがあったらIssueで報告をお願いします。  
・このプロジェクトはStoneblock4のVer1.5.0用に作られたものです翻訳です。最新バージョンに更新するに当たって、正しく使えなくなる場合があります。もしそうなった場合はIssueで報告をお願いします。
